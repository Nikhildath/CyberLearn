# **App Name**: CyberLearnHQ

## Core Features:

- Lesson Display: Display comprehensive lessons on key cybersecurity topics such as Phishing and Malware.
- Progress Tracking: Implement a progress bar to visually track user advancement through lessons.
- AI Assistant: Integrate Text-to-Speech (TTS) and Speech-to-Text (SST) for interactive AI responses to cybersecurity questions. The LLM acts as a tool for providing guidance and information, helping students understand complex concepts.
- Simulated Authentication: Provide a simulated login feature with username 'nikhil' and password 'nikhil'.
- Topic Expansion: Add 8 more Cybersecurity topics beyond phishing and malware

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey trust and security.
- Background color: Light blue (#D2E3FC), a lighter desaturated tint of the primary color for a clean, professional look.
- Accent color: Teal (#2E927D) for interactive elements and key highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clear and recognizable icons to represent different cybersecurity concepts.
- A clean, intuitive layout that facilitates easy navigation and learning.
- Subtle animations to provide feedback and enhance user engagement.
